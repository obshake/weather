# weather
A sleek, real-time weather application that provides instant weather updates and forecasts. Built using clean Vanilla JavaScript and the OpenWeatherMap API.
<br>
## Features

Live Weather Data: Fetches current temperature, humidity, and wind speed.

Search Autocomplete: Includes a custom-built suggestion list for global cities.
